KI-Anbieter & API-Key
Dein eigener Key: Der API-Key wird ausschließlich in deinem Browser gespeichert und für AI-Funktionen (E-Mail-Import, PDF, HKCM) verwendet.
Wo bekomme ich einen API-Key?
Claude (Anthropic)
→ console.anthropic.com
→ API Keys → Create Key → beginnt mit sk-ant-
GPT (OpenAI)
→ platform.openai.com/api-keys
→ Create new secret key → beginnt mit sk-
Gemini (Google)
→ aistudio.google.com/app/apikey
→ Get API key → beginnt mit AIza
⚠ Nur AI Studio Keys haben ein kostenloses Kontingent. Google Cloud Console Keys haben limit:0 und funktionieren nicht ohne Abrechnung.
Der Key wird im localStorage gespeichert (getrennt von Investitionsdaten). Er verlässt deinen Browser nur verschlüsselt via HTTPS. Keine Drittpartei hat Zugriff.
